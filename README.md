# pics-to-gif
pics to gif / 照片转gif动图


## install
```
go get github.com/Chyroc/pics-to-gif
```

## use
```
NAME:
   pics-to-gif - 图片转GIF动图

USAGE:
   t -f folder [-o out_filename] [-p goroutine]

VERSION:
   v0.1.0

ARGS:
     -f   图片所在文件夹
     -o   输出GIF文件名
     -p   并发数

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version
```
